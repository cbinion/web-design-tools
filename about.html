What have learned in this class?

HTML/CSS

I learned a lot of fundamental and some advanced concepts related to HTML and CSS:

  • Laying out a website using semantically correct and accessibility-friendly tags throughout the entire website
  • Optimizing images in the photo gallery to display quicker and ease the load of the entire website
  • Creating forms on the contact page
  • Working with layout of elements on a page with floats
  • Utilizing a CSS design library with W3.CSS

General Web Development

I was able to gain some essential knowledge regarding developing a website:

  • Editing files and utilizing extensions within an integrated development environment (IDE) with Visual Studio Code
  • Optimizing images to display to quicker and ease the load of the entire website
  • Updating a website using file transfer protocol (FTP)
  • Troubleshooting when errors occur

WordPress

Learning how to work with a content management system (CMS) provides endless possibilities for creating an easily maintainable website:

  • Setup and customization of a base WordPress website
  • Installing and activating plugins with a cookie banner, contact form, and backup plugins
  • Applying themes and customizing their look and feel
  • Creating static pages and organizing posts to display on them
